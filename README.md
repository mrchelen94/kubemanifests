# О проекте

#### Веб-приложение представляет собой простенький "todo list", в котором можно добавлять задачи. Записи добавляются в бэкенд на _node.js_, который подключается к NOSql БД MongoDb и соответственно создается запись. В качестве веб-интерфейса используется _nginx_ для взаимодействия с сайтом

# Состав

#### В директории backend находится Dockerfile с образом бэкенда приложения на основе _node.js_ и файлом _index.js_, к котором располагается код бэкенда. Сам образ залит на мой dockerhub(_oracleguy04/k8snodejs:1.0_)
#### NoIngress_webapp содержит готовые манифесты kubernetes для установки веб-приложения без ингресса с использованием proxy_pass nginx
#### ingress_webapp содержит манифесты веб-приложения и манифесты для установки ingress контроллера nginx-ingress:
#### - Манифесты с расширением *.yaml - манифесты для установки nginx-ingress;
#### - Манифесты с расширением *.yml - манифесты для установки веб-приложения;

# Установка в кластер kubernetes

#### Перед разворачиванием веб-приложения необходимо в файле _hosts_ ОС присвоить ip-адрес, где развернут **kubernetes**, для доменного имени ```"task-list.com"``` и сделать проброс для порта 30090 на кластер кубернетеса.
#### Далее необходимо применить манифесты:
#### - kubectl apply -f deploy/NoIngress_webapp - для развертки веб приложения без ingress;
#### - kubectl apply -f deploy/ingress_webapp - для развертки веб приложения с ingress.
#### После выполненных действий заходим в браузере по адресу ```"http://task-list.com:30090"``` и получаем наслаждение :)
